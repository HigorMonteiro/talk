# Apresentação - Desenvolvimento Frontend

Apresentação criada com [Slidev](https://sli.dev/).

## Como usar

### Opção 1: Usando npx (recomendado)

```bash
npx @slidev/cli slides.md
```

### Opção 2: Usando npm scripts

```bash
npm run dev
```

Isso iniciará o servidor de desenvolvimento. Acesse `http://localhost:3030` no navegador.

## Build

Para gerar uma versão estática:

```bash
npm run build
```

## Exportar

Para exportar como PDF ou PNG:

```bash
npm run export
```

## Navegação

- **Próximo slide**: `→`, `Space`, `Enter`, `PageDown`
- **Slide anterior**: `←`, `Backspace`, `PageUp`
- **Primeiro slide**: `Home`
- **Último slide**: `End`

## Estrutura

- `slides.md` - Arquivo principal com todos os slides em Markdown
- `package.json` - Configuração do projeto
